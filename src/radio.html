  3. Radios

Basic Radio

You can use the Radio component to render a checkbox input.

Radio Colors

Use the color prop to change the color of the checkbox. You can use the following colors: primary, secondary, success, danger, warning, info, dark. Default color is primary.

Radio Sizes

Use the radioSize prop to change the size of the checkbox. You can use the following sizes: sm, md, lg. Default size is sm.

Custom Radio

Use RadioGroup component from headless UI to create a custom radio. Checkout the Headless UI documentation documentation for more details.
