Basic Button

The .btn classes are designed to be used with the</button> element. However, you can also use these classes on </a> or <input> elements.

Link

Color Buttons

Use color modifier classes to change the button style. Example: .btn-primary, .btn-secondary, .btn-success, .btn-danger, .btn-warning, .btn-info and .btn-dark

Button Sizes

Use size modifier classes to change the button size. Example: .btn-xs, .btn-sm, .btn-md, .btn-lg, .btn-xl. Default is .btn-md

Outline Buttons

Use default classes with the .btn-outline-* ones to create an outline button.

Soft Buttons

Use default classes with the .btn-soft-* ones to create a soft button.

Plain Buttons

Use default classes with the .btn-plain-* ones to create a plain button.

Rounded Buttons

Use .rounded-full class to create a rounded button.

Icon Buttons

By adding icon to the button, you can create icon buttons

Loading Buttons

By adding a loading icon with disabled attribute, you can create loading button.

Disabled Buttons

To disable a button, use disable="true" attribute on it.

Block Buttons

Use .w-full class to create a block button.

Group Buttons

To create button groups, wrap all the buttons with a .btn-group class.
